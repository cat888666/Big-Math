# 分子有理化(**恒等变形**)

$\displaystyle\lim_{ x \to 0y \to 0}\frac{2-\sqrt{ xy+4 }}{xy}$

$$\displaystyle\lim_{ x \to 0y \to 0}\frac{2-\sqrt{ xy+4 }}{xy}\to \displaystyle\lim_{ x \to 0y\to 0 } \frac{(2-\sqrt{ xy+4 })(2+\sqrt{ xy+4 })}{xy(2+\sqrt{ xy+4 })}\to \displaystyle\lim_{ x \to 0y\to 0   } \frac{-1}{2+\sqrt{ xy+4 }}$$
# 整体代换+洛必达

$\displaystyle\lim_{ x \to 0y \to 0}\frac{2-\sqrt{ xy+4 }}{xy}$
$$
\begin{align*}
令xy=t，t\to 0 \qquad\\  \\
原式=\displaystyle\lim_{ t \to 0 } \frac{2-\sqrt{ t+4 }}{t}\quad\\  \\
洛必达得\displaystyle\lim_{ t \to 0 } \frac{0- \dfrac{1}{2\sqrt{ t+4 }}}{1} \\
\end{align*}
$$
