# 正项级数收敛性判断
对正项级数：$\displaystyle\sum_{n=1}^\infty u_{n}$
$\displaystyle L=\lim_{ n \to \infty  } \frac{u_{n+1}}{u_{n}}或\displaystyle \lim_{ n \to \infty  }\sqrt[n]{ u_{n} }$
如果$L<1$，级数收敛
如果$L>1$，级数发散
如果$L=1$，需要额外判断
$$
\begin{align*}
\lim_{ n \to \infty } \frac{u_{n}}{ \frac{1}{n^p}}=L\qquad \begin{cases} p\leq 1,0< L\leq \infty \qquad级数发散\\ \\
p>1,0\leq L< \infty \qquad级数收敛
\end{cases}
\end{align*}
$$

### 经典结论

$\displaystyle\lim_{ n \to \infty  }\sqrt[n]{ a }=1$
$\displaystyle\lim_{ n \to \infty  }\sqrt[n]{ n }=1$
$\displaystyle\lim_{ n \to \infty  }\sqrt[n]{n^{2}-3n }=1$ (里面是任何多项式都可)
$\displaystyle\lim_{ n \to \infty  }\sqrt[n]{ n! }=+\infty$

# 交错级数收敛判别
对交错级数$\displaystyle\sum_{n=1}^\infty(-1)^n u_{n}或\sum_{n=1}^\infty(-1)^{n+1}u_{n}$（$u_{n}$为正项）
如果$\displaystyle \lim_{ n \to \infty }u_{n}=0且数列u_{n}递减$
则原级数收敛 ^a0219e

对于任意级数$\displaystyle\sum_{n=1}^\infty a _{n}$，若$\displaystyle\sum_{n=1}^\infty |a_{n}|$收敛，称原级数**绝对收敛**
若$\displaystyle\sum_{n=1}^\infty a _{n}$收敛，$\displaystyle\sum_{n=1}^\infty |a_{n}|$发散，称原级数**条件收敛**
